▸ 折现率规则说明（CROSS）

① 基础利率曲线（负债，即期）

按期限 t（年）分三段拼接：

0 < t ≤ 20750 日移动平均国债收益率曲线
20 < t ≤ 40终极利率过渡曲线（二次插值平滑过渡）
t > 40终极利率（UFR）4.5%

② 终极利率过渡曲线（二次插值，完整）

先对 20 < t ≤ 40 段，将市场曲线第 20 年值 r*20 线性外推至终极利率 UFR，得：

rt = r*20 + (UFR − r*20) · (t − 20) / (40 − 20)

再与市场曲线 r*t 按年期权重混合，得到基础利率 Rt（因含 (t−20)² 项，故称二次插值）：

Rt = r*t · (40 − t) / (40 − 20) + rt · (t − 20) / (40 − 20)

③ 综合溢价（负债，前 20 年分档）

高利率保单1999 年及以前签发 · 75BP
万能/投连/中短存续30BP
其他产品45BP

20 < t ≤ 40 线性衰减至 0；t > 40 为 0。

④ 利率风险情景曲线（基础情景 / 不利情景）

基础情景基础利率曲线为即期，同样分三段：0 < t ≤ 20 为 60 日移动平均国债收益率曲线；20 < t ≤ 40 为终极利率过渡曲线（二次插值，同②）；t > 40 为 UFR 4.5%。

不利情景（上升 / 下降）基础利率曲线：

0 < t ≤ 20： rs,t = rt · (1 + Stresst)   |   t > 40： rs,t = UFR · (1 + Stress40)

其中 rt 为基础情景即期利率；20 < t ≤ 40 段采用二次插值（同②，基于受压曲线）。UFR = 4.5%，Stress40 上升 = 17%、下降 = −11%。

不利情景压力参数 Stresst：前 20 年与 40 年后的参数见下表；20 < t ≤ 40 按线性插值：

Stresst = Stress20 · (40 − t) / (40 − 20) + Stress40 · (t − 20) / (40 − 20)

⑤ 折现率换算

即期折现率 = 基础利率曲线 + 综合溢价；远期折现率由即期折现率曲线换算得到。负债曲线采用 MA750；利率风险情景（基础 / 上升 / 下降）以 MA60 施加监管压力后外推得到。

依据：《保险公司偿付能力监管规则第 3 号：寿险合同负债评估》第十九条及附件 1；《保险公司偿付能力监管规则第 8 号：市场风险最低资本》第十九条、第二十条及附件 3。

▸ 折现率规则说明（Solvency II · Smith-Wilson）

① 基础利率曲线（负债 / 利率，即期）

取国债即期收益率曲线当日值（不再使用移动平均），对流动点的年化远期利率做 Smith-Wilson 拟合，外推至 120 年收敛于终极利率 UFR。

fwd(t) = UFR + Σᵢ ζᵢ · W(t, uᵢ)， W(t,u)=e^{−α(t+u)}[α·min(t,u)−e^{α·min}+e^{αt}+e^{αu}−e^{α(t+u)}] ⁄ [α·e^{α·min}·(e^{αt}−1)(e^{αu}−1)]

α 由 EIOPA 收敛条件确定：取最小 α≥α下限 使 |fwd(CP)−UFR|≤0.10bp（CP = 40（默认 LLP+20 = 40））。UFR 上/下情景分别为利率风险不利参数下的终极利率。

② 综合溢价（负债 / 利率）

以远期平移方式叠加：fwd′t = fwdt + 溢价。负债高 / 中 / 低三档溢价独立可调（具体数值见下方参数面板）；利率曲线（基础 / 上升 / 下降）采用「终极利率溢价 P10」做远期平移。UFR 上升 / 下降 = 基础 UFR ×(1+40年不利幅度)。

③ 利率风险情景（上升 / 下降）

对国债即期曲线施加监管压力参数：受压即期 = 即期×(1+压力参数)，再做 Smith-Wilson 拟合。压力参数如下：

利率向上 / 向下压力参数（年，%）
年 t上升下降年 t上升下降

不利情景下的综合溢价保持与基础情景下的综合溢价一致。

④ 输出

一套方案同时输出：负债折现 高 / 中 / 低 + 利率曲线 基础 / 上升 / 下降，年期 1–120 年；即期由含溢价远期曲线还原。格式与CROSS汇总表一致。
